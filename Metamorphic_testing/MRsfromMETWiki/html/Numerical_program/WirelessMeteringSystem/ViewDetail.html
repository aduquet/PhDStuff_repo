Wireless metering system

Tag: Wireless communication embedded systems
Edit   Star   Delete   Add Picture

Program Information

Name: Wireless metering system
Domain: Numerical program
Functionality: A wireless metering system used in two electricpower companies.And in this project, we test the meterreadingfunction enabled by the software inside the RF module.
Input: The signal power to antenna (that is, Pant)(Antenna is used to receive and radiate the RF signal.) Low Noise Amplifier (LNA) is a circuit in the RF receiver to amplify the input signal (PRX_IN) to get a higher but low-noise output signal (PRX_OUT) (as to reduce the Bit Error). Power Amplifier (PA) is a circuit in the RF transmitter to amplify the input signal (PTX_IN) to get a higher output signal (PTX_OUT) for transmission. 
Output: The computed Received Signal Strength Indicator (RSSI). The relationship between the RSSI and the signal power PRX_OUT can be expressed by the following equation:
PRX_OUT=32RSSI−3VAG_SETTING−RSSI_Offset
Note that the gain of LNA (GLNA) can be expressed as:
GLNA=PRX_OUT−Pant
Thus, we can derive the following Equation:
RSSI=23(PRX_OUT+3VGA_SETTING+RSSI_Offset=23(GLNA+Pant+3VGA_SETTING+RSSI_Offset))
GLNAVGA_SETTING and RSSI_Offset are the parameters set by RF-Soft to configure the meter reading process. 

Reference

 Testing Embedded Software by Metamorphic Testing: a Wireless Metering System Case Study https://doi.org/10.1109/LCN.2011.6115306
