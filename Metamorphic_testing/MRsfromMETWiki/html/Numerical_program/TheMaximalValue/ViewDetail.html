The Maximal Value

Tag:
Edit   Star   Delete   Add Picture

Program Information

Name: The Maximal Value
Domain: Numerical program
Functionality: Calculates the maximal value from a set of input data
Input: Suppose that there is a set of real numbers, X={x1,x2,⋯,xn}.And there is another set X′. 
Output: The maximal value of X is m,and the maximal value in X′ is m′.

Reference

Teaching an End-User Testing Methodology https://doi.org/10.1109/CSEET.2010.28

MR Information

MR1------

Description:
Property: X′=X∪{a} where a>m ⇒ m′>m
Source input: X
Source output: m
Follow-up input: X′
Follow-up output: m′
Input relation: X⇒X′ where X′=X∪{a} and a>m
Output relation: m′>m
Pattern:

MR2------

Description:
Property: X′=X∪{b} where b<m ⇒ m′=m
Source input: X
Source output: m
Follow-up input: X′
Follow-up output: m′
Input relation: X⇒X′ where X′=X∪{b} and b<m
Output relation: m′=m
Pattern:

MR3------

Description:
Property: X′=X−{m} (deleting all elements that are equal to m) ⇒ m′<m
Source input: X
Source output: m
Follow-up input: X′
Follow-up output: m′
Input relation: X⇒X′ where X′=X−{m}(constructed by deleting all elements that are equal to m)
Output relation: m′<m
Pattern:

MR4------

Description:
Property: X′={0.0−x1,0.0−x2,⋯,0.0−xn}⇒m′=c where c is the minimal value in X 
Source input: X
Source output: m
Follow-up input: X′
Follow-up output: m′
Input relation: X⇒X′ where X′={0.0−x1,0.0−x2,⋯,0.0−xn}
Output relation: m′=c where c is the minimal value in X

MR5------

Description:

Source input: X
Source output: m

Follow-up output: m′
Input relation: X⇒X′ where X′ is constructed by changing the order of the data in X
Output relation: m′=m
