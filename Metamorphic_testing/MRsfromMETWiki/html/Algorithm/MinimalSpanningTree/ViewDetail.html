Program Information

Name: Minimal Spanning Tree
Domain: Algorithm
Functionality: Using Kruskal's Algorithm to find minimal spanning tree.
Input: The below diagram figure 1.(a) shows that an undirected non-negative weighted graph with vertices set V={a,b,c,d,e,f,g,h,i} and edge set E as
t={(a,b),(b,c),(b,h),(a,h),(b,c),(c,i),(i,h),(h,g),(i,g),(c,f),(g,f),(c,d),(d,f),(d,e),(e,f)}
.  Consider the figure 1.(a) as a base model graph G={V,E}, whereas |V|=9 and |E|=14.Minimal spanning tree for the same is shown in figure 1.(b) as a sub graph G={V′,E′},whereas |V|=9 and |E|=8.    
Output: Figure 2.(b) displays the obtained Kruskal's minimal spanning tree as
t′={(a,b),(a,h),(c,i),(h,g),(c,f),(g,f),(c,d),(d,e)}
.

Reference

 Adopting Metamorphic Relations to verify NonTestable Graph Theory Algorithms https://doi.org/10.1109/ICACCE.2015.138
