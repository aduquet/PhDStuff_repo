Angiogenesis model

Tag: ABM cancer model Simulation Models  
Edit   Star   Delete   Add Picture

Program Information

Name: Angiogenesis model
Domain: Algorithm
Functionality: The simulation model studies angiogenesis, the growth of new blood vessels that is crucial for tumors to be able to grow enough to cause problems within their tissue.(This model's MRs are derive accroding to the types of metamorphic properties for ABM.You can find it by searching ``ABM'')
Input: In this model there are tumor, normal,and vascular(blood vessel) cells represented as agents that exist within a three-dimensional non-toroidal lattice.
Output: To validate this complex multi-scale model.

Reference

 Metamorphic Validation for Agent-based Simulation Models https://dl.acm.org/citation.cfm?id=3015607 

MR Information

These relations are identified based on metamorphic properties that exist in angiogenesis, which in turn were defined using the categories listed in ``ABM". They define the expected behavior change due to the listed change for a given parameter or system property.
