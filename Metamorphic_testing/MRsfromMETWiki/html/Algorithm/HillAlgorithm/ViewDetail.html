Hill algorithm

Tag: encryption programs Hill algorithm
Edit   Star   Delete   Add Picture

Program Information

Name: Hill algorithm
Domain: Algorithm
Functionality: It encrypts a plaintext, which is usually a sequence of m letters, into a ciphertext that has the same size as the plaintext.
Input: The input is defined as a tuple ⟨P,K⟩,where P and K refer to a plaintext and a key matrix, respectively. The source test case is denoted as (P,K),the follow-up test case is denoted (P′,K′).
Output: The outputs are defined as Ch and C′h correspondingto the ciphertext.

Reference

 A property-based testing framework for encryption programs https://doi.org/10.1007/s11704-014-3040-y
