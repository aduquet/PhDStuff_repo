Edit   Star   Delete   Add Picture

Program Information

Name: Tot_info
Domain: Algorithm
Functionality: Printing the Kullbacks information measure, the degree of freedom and the possibility density of distribution of one table.
Input: T: N tables(Type: Table)
Output: For each table tiT, Ii: The Kullbacks information measure(Type: ?) Di: The degree of freedom(Type: Integer) Qi: the possibility density of distribution of one table(Type: ?)

Reference


Metamorphic slice: An application in spectrum-based fault localization
http://dx.doi.org/10.1016/j.infsof.2012.08.008

MR Information

MR1------

Description:
Property:
Source input: Ts
Source output: Isi,Dsi,Qsi
Follow-up input: Tf
Follow-up output: Ifi,Dfi,Qfi
Input relation: Tf=Ts∪Ts
Output relation: (totI)f = 2×(totI)s, where totI is the summaries of all Ii. (totD)f = 2×(totD)s, where totD is the summaries of all Di. Qf=Qs
Pattern:

MR2------

Description:
Property:
Source input: Ts
Source output: Isi,Dsi,Qsi
Follow-up input: Tf
Follow-up output: Ifi,Dfi,Qfi
Input relation: Rfi=2×Rsi Cfi=Csi where Ri is the number of rows of ti, Ci is the number of columns of ti ,and the content from the (Rsi+1)th row to the (2×Rsi)th row in tfi is the duplicate of its first rsi rows.
Output relation: Ifi=2×Isi Dfi=Dsi+Rsi×(Csi−1) Qf=Qs
Pattern:

MR3------

Description:
Property:
Source input: Ts
Source output: Isi,Dsi,Qsi
Follow-up input: Tf
Follow-up output: Ifi,Dfi,Qfi
Input relation: Tf = Each element of Ts multiplied by k.
Output relation: Ifi=k×Isi Dfi=Dsi Qf=Qs
