Program Information

Name: MultipleKnapsack
Domain: Optimization algorithms
Functionality: Solving the so- called multiple knapsack problem.
Input: P(p1,p2,⋯,pn): The profit of items(Type: Set) W(w1,w2,⋯,wn):The weight of items(Type: Set) C(c1,c2,⋯,cm):The capacity of knapsacks(Type: Set)
Output: T:The solution of the problem(Type: Integer) Y(y1,y2,⋯,yn):An array of the solution(Type: Array)

Reference

 
How Effectively Does Metamorphic Testing Alleviate the Oracle Problem?
http://dx.doi.org/10.1109/TSE.2013.46
