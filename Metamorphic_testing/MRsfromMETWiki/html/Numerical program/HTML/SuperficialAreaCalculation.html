Superficial Area Calculation

Tag: superficial area calculation GIS DEM TIN
Edit   Star   Delete   Add Picture

Program Information

Name: Superficial Area Calculation
Domain: Numerical program
Functionality: A superficial area calculation program, which is common for most of the GIS systems.
Input: Coordinate (X,Y,Z),and the corresponding follow-up test case (X'.Y',Z').
Output: The area denote as S(X,Y,Z).

Reference

 Metamorphic Testing of a Superficial Area Calculation Program  https://doi.org/10.1109/QRS-C.2016.64

MR Information

MR1------ Construction MRs with geometric properties

Description: If the target domain rotates around Z axis,the superficial area holds in line.
Property: ∀i((X′i=Xcosθ−Ysinθ)∧(Y′i=Xsinθ+Ycosθ)∧(Z′i=Z))⇒S(X,Y,Z)=S(X′,Y′,Z′)
Source input: (X,Y,Z)
Source output: S(X,Y,Z)
Follow-up input: (X′,Y′,Z′)
Follow-up output: S(X′,Y′,Z′)
Input relation: ∀i((X′i=Xcosθ−Ysinθ)∧(Y′i=Xsinθ+Ycosθ)∧(Z′i=Z))
Output relation: S(X,Y,Z)=S(X′,Y′,Z′)
Pattern:

MR2------ Construction MRs with geometric properties

Description: If a target domain translates in vector (dx,dy),the superficial area of it remain unchanged.
Property: ∀i((X′i=Xi+dx)∧(Y′i=Yi+dy)∧(Z′i=Zi))⇒S(X,Y,Z)=S(X′,Y′,Z′)
Source input: (X,Y,Z)
Source output: S(X,Y,Z)
Follow-up input: (X′,Y′,Z′)
Follow-up output: S(X′,Y′,Z′)
Input relation: ∀i((X′i=Xi+dx)∧(Y′i=Yi+dy)∧(Z′i=Zi))
Output relation: S(X,Y,Z)=S(X′,Y′,Z′)
Pattern:

MR3------ Construction MRs with mathematical properties

Description: As the target area is composed of several different grids, the sequence of the grids would not affect the computation result  of  superficial  area  calculation. 
Property: ∀i((X′i=Xn+1−i)∧(Y′i=Yn+1−i)∧(Z′i=Zn+1−i))⇒S(X,Y,Z)=S(X′,Y′,Z′)
Source input: (X,Y,Z)
Source output: S(X,Y,Z)
Follow-up input: (X′,Y′,Z′)
Follow-up output: S(X′,Y′,Z′)
Input relation: ∀i((X′i=Xn+1−i)∧(Y′i=Yn+1−i)∧(Z′i=Zn+1−i))
Output relation: S(X,Y,Z)=S(X′,Y′,Z′)
Pattern:

MR4------ Construction MRs with mathematical properties

Description: As the target area is composed of several different grids, the sequence of the grids would not affect the computation result  of  superficial  area  calculation. 
Property: ∀i((X′i=k⋅Xi)∧(Y′i=k⋅Yi)∧(Z′i=k⋅Zi))⇒S(X′i,Y′i,Z′i)=k2⋅S(Xi,Yi,Zi)
Source input: (X,Y,Z)
Source output: S(X,Y,Z)
Follow-up input: (X′,Y′,Z′)
Follow-up output: S(X′,Y′,Z′)
Input relation: ∀i((X′i=k⋅Xi)∧(Y′i=k⋅Yi)∧(Z′i=k⋅Zi))
Output relation: S(X′i,Y′i,Z′i)=k2⋅S(Xi,Yi,Zi)
Pattern:

MR5------ Construction MRs with focusing properties of DEM

Description: Let S(Gi) be the superficial area of the mesh grid Gi. The precision of G3 is two times of G2, which precision is two times of G1. 
Property: G1⊂G2⊂G3⇒S(G1)≤S(G2)≤S(G3)
Source input: G1
Source output: S(G1)
Follow-up input: G2,G3
Follow-up output: S(G2),S(G3)
Input relation: G1⊂G2⊂G3
Output relation: S(G1)≤S(G2)≤S(G3)
Pattern:

MR6------ Construction MRs with algorithm of TIN

Description: (a,b,c) is the edges of a triangle.
Property: (a′=2×c2+2×b2−a2−−−−−−−−−−−−−−−√,b′=b,c′=c)⇒S(a′,b′,c′)=S(a,b,c)
Source input: (a,b,c)
Source output: S(a,b,c)
Follow-up input: (a′,b′,c′)
Follow-up output: S(a′,b′,c′)
Input relation: (a′=2×c2+2×b2−a2−−−−−−−−−−−−−−−√,b′=b,c′=c)
Output relation: S(a′,b′,c′)=S(a,b,c)
Pattern:

MR7------ Construction MR with the characters of superficial area

Description: For a target domain, if it could be divided into two parts, then the superficial area of the domain is the total of the sub parts.
Property: X′={X0,X1,⋯,Xm}∧Y′={Y0,Y1,⋯,Ym}∧Z′={Z0,Z1,⋯,Zm}∧X′′={Xm,Xm+1,⋯,Xn−1,X0}∧Y′′={Ym,Ym+1,⋯,Yn−1,Y0}∧Z′′={Zm,Zm+1,⋯,Zn−1,Z0}⇒S(X,Y,Z)=S(X′,Y′,Z′)+S(X′′,Y′′,Z′′)(1<m<n)
Source input: (X,Y,Z)
Source output: S(X,Y,Z)
Follow-up input: (X′,Y′,Z′),(X′′,Y′′,Z′′)
Follow-up output: S(X′,Y′,Z′),S(X′′,Y′′,Z′′)
Input relation: (X′={X0,X1,⋯,Xm}∧Y′={Y0,Y1,⋯,Ym}∧Z′={Z0,Z1,⋯,Zm}∧X′′={Xm,Xm+1,⋯,Xn−1,X0}∧Y′′={Ym,Ym+1,⋯,Yn−1,Y0}∧Z′′={Zm,Zm+1,⋯,Zn−1,Z0}
Output relation: S(X,Y,Z)=S(X′,Y′,Z′)+S(X′′,Y′′,Z′′)(1<m<n)
