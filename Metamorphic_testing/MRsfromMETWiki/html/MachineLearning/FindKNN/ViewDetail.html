Program Information

Name: FindKNN
Domain: Machine learning
Functionality: To find k nearest neighbors of P(one m-dimensional test point) from S(n m-dimensional sample points).
Input: P=(p1,p2,⋯,pm):One m-dimensional test point(Type: Point) S=(s1,s2,⋯,sn) where si=(si1si2,⋯,sim):N m-dimensional sample points(Type: Set) k:A positive integer(Type: Integer)
Output: O=(o1,o2,⋯,ok) :Indices of the k nearest neighbors(Type: Index)

Reference


          How Effectively Does Metamorphic Testing Alleviate the Oracle Problem? http://dx.doi.org/10.1109/TSE.2013.46
